# Great Moove Librery

## Description

to start working on The Great Move Library project you have to:

- Implement of the provided databases.
- Start a PHP server with the command `php -S localhost:9000`.
- Logging in as an admin using the following credentials:
  - Admin Account: admin
  - Password: admin123
- Add teacher emails and student emails.
- Add users.
- Add books.
- Add courses.
- Manag pending items with the admin profile.
- Manag uploads and users with the admin profile.

**Note:** All data will be uploaded to the "uploads" folder provided in the server.

Thank you for choosing Great Move Library! If you have any questions or need assistance, please don't hesitate to reach out.
